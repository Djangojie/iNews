# iNews 新闻门户网站

🖐 Hi，欢迎来到我们的代码仓库 

本项目采用技术栈如下

前端
- 主框架：React (全面使用 hooks)、React-Router6、Context
- 脚手架：create-react-app
- UI 框架：Ant Design
- CSS 解决方案：styled-components
- 动画库：ant-motion，css-transition-group，particles-bg
- 网络请求: axios
- 异步解决方案：Promise
- 协作工具：Git
- 函数工具库：loadsh

后端
Node.js + 轻服务

如需使用本项目，可 `git clone` 克隆到本地

执行 `yarn` 或者 `yarn add` 命令安装依赖

执行 `yarn start` 启动项目
